# 无限画布 Agent 转发器

把无限画布 Agent 当前对话或最后一条回复一键发送到 ChatGPT、Gemini；也可以作为画布生图桥接器，把提示词和参考图发送到 ChatGPT / Gemini，生成完成后自动回填到无限画布。扩展复用浏览器中已有的登录状态，不读取账号密码和 API Key。

## 本地安装

1. 打开 `chrome://extensions/`。
2. 开启右上角「开发者模式」。
3. 点击「加载已解压的扩展程序」。
4. 选择本目录 `chrome-extension/agent-share`。

## 使用

### 转发 Agent 对话

1. 在无限画布中打开 Agent 对话。
2. 点击浏览器工具栏里的「无限画布 Agent 转发器」。
3. 选择「整个当前对话」或「最后一条 Agent 回复」。
4. 点击 ChatGPT 或 Gemini，扩展会打开目标网站、填入内容并自动发送。

### 桥接图片生成

1. 在无限画布的图片生成器里选择 `ChatGPT Image（浏览器插件）` 或 `Gemini Image（浏览器插件）`。
2. 输入提示词，可附带参考图。
3. 点击生成后，扩展会打开目标网站、上传参考图、填入提示词并发送。
4. 参考图上传成功后才会发送提示词；如果上传失败，会停止发送并提示。
5. 目标网站生成出新图片后，扩展会尝试自动读取图片并回填到当前画布节点；若自动回填失败，可在 ChatGPT/Gemini 页面右下角点击「回填到 DOT AI」手动回填。

## 说明

- Agent 对话转发仍只转发文字内容，不上传 Agent 对话中的图片附件。
- 生图桥接会依次尝试文件 input、粘贴、拖拽三种参考图上传方式，并在发送提示词前检测新增参考图。
- 生图桥接依赖 ChatGPT / Gemini 页面结构和浏览器已登录状态；目标网站调整输入框、上传控件或结果图结构后，可能需要同步更新 `destination.js` 中的选择器。
- 待发送内容仅暂存在 `chrome.storage.local`，成功点击发送或回填后删除，超过 15 分钟自动过期。
